第１回 WWW と HTML

WebページはHTML、CSS、JavaScript、画像・音声・動画の４つの要素によって構成される

HTMLについて

・ブロック要素とインライン要素

HTMLは要素によって構成される。要素はブロック要素(例：段落<p>や見出しなど)とインライン要素 (例：<strong>やリンクなど)の２つに分けることができる。他にも空要素による段落分けや要素への属性の追加が行える

各要素について

  1. <!DOCTYPE html>：「お約束」となる記述
  2. <html>：文書全体を指す要素。これもお約束
  3. <head>：ページには直接表示されない要素を表す
  4. <meta charset="utf-8">：文書内で使用する文字コードを設定する要素
  5. <title>：Webページのタイトルを設定する

    演習の成果：演習1-4のHTMLファイル

第2回 CSS による Web ページの装飾

・CSSとは

CSSはWebページを構成する要素の１つであり、HTMLを装飾する役割がある。

CSSにはテキストの色変更や背景色の追加など様々な装飾を加えることができ、それによってより読みやすく見た目の良いWebページの作成が可能。

CSSは装飾したい要素を指定するセレスタ装飾の種類を指定するプロパティ名装飾の具体的な値を指定するプロパティ値からなる

・CSSの例

  1. (color)：テキストの色変更を行える
  2. (font-weight)：テキストの太さが変更できる
  3. (text-shadow)：テキストに影を追加できる
  4. (font-family)：テキストのフォントを変更できる
  5. (border-bottom)：下線を追加できる

これらはほんの一例で他にも多様な種類のCSSによって装飾を行える

    演習の成果：演習2-3のHTMLファイル

第3回 JavaScript の基本

JavaScriptとは

JavaScriptとはプログラミング言語の一種でWebページ上にさまざまな機能を追加する。

API

ブラウザにはJavaScriptを利用した機能(API)がある。

・APIの例

・ブラウザAPI

  1. DOM API：プログラムを通じたHTML要素や属性、CSS などの追加・編集・削除を可能にする
  2. Geolocation API：プログラムを通じて現在位置を取得する
  3. GanvasとWebGL：2Dと3Dのグラフィックを扱うプログラミングを可能にする
  4. WebRTCリアルタイムの通信やビデオを用いた遠隔会議を可能にする

・サードパーティAPI

  • Twitter API：ツイートの検索と表示
  • Google Map API：Webページにマップを貼り付けられる

    • インタプリタ言語とコンパイル言語

    名前の似ているJavaとJavaScriptだが大きな違いとしてインタプリタ言語かコンパイル言語かの違いがある。

    コンパイル言語

    Javaはコンパイル言語。Javaはプログラムを実行する際にjavacでコンパイル→javaで実行という2段階の手順を踏む必要がある。

    インタプリタ言語

    JavaScriptはインタプリタ言語。JavaScriptプログラムはそのまま１行ずつ読み取りながら実行される為、Javaにおけるコンパイルの過程が省略されている

    インタプリタ言語は実行の際の手間が少なく実行、デバックの行いやすさが長所である一方、エラーの発見、原因の究明が行いにくいという欠点もある。
